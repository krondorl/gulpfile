# gulpfile
My Gulp workflow, 2016-08-24

Using:
* Gulp
* Sass
* PostCSS
  * Autoprefixer
  * CSSNano
* Babel
* Babel Preset 2015

## 1. Install process
`npm i gulp gulp-postcss gulp-sass gulp-babel babel-preset-es2015 autoprefixer cssnano`

## 2. Directories
* `./scss`, source styles
* `./js`, source scripts
* `./dist`, compiled assets

## 3. Watch
`gulp`
